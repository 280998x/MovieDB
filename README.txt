Para este proyecto se utilizaron 2 objetos,’Movie’ para guardar la información de las películas en memoria e ‘ImageCache’ para guardar las imágenes en el cache

- HomeViewController

	Esta clase es encargada de la vista inicial de la app, iniciativa la colección de 
	vistas para las películas y prepara la siguiente vista en caso de seleccionar una 
	película

- MoviesViewController

	Esta clase es encargada de obtener los datos de las películas, así como de dibujar 
	las vistas

- MovieDetailsController

	Esta clase es encargada de mostrar la información de una película cuando la vista 	es seleccionada
